# Student Grade Management System

This is a C++ project developed for the STL and Basic Data Structures self-learning assignment. It implements a student grade management system using standard C++ templates and the Standard Template Library (STL).

## Features
1. Add student data (with duplicate ID check)
2. List all students with formatted layout
3. Sort students by score in descending order
4. Search student by ID
5. Show grade statistics (Average, Max, Min, Pass/Fail count) using custom Template Functions

## Requirements
- Compiler: `g++` supporting **C++17** standard or higher.

## How to Compile and Run
Open your terminal in the project directory and run the following commands:

```bash
# Compile the program
g++ -std=c++17 main.cpp -o main

# Run the program
./main
